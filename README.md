## 🚀 Project Overview
This is a **mini full-stack web application** built with **React (frontend)** and **Spring Boot (backend)**.  
It demonstrates a full-stack setup with REST APIs, dashboard functionality, and optional JWT authentication.

## 🛠️ Features
- Interactive dashboard for user data  
- CRUD operations through REST APIs  
- Responsive design for desktop and mobile  
- Optional JWT-based authentication  
- Swagger API documentation for easy API exploration  

## 💻 Tech Stack
- **Frontend:** React  
- **Backend:** Spring Boot, Java, Spring Security (optional JWT)  
- **Database:** H2 / MySQL / PostgreSQL (as configured)  
- **Deployment:** Vercel (frontend), Render (backend)
  
## 🔗 Usage
1. Open the **frontend live link** in a browser to interact with the dashboard.  
2. Explore all available API endpoints via **Swagger documentation**.  
3. Perform CRUD operations and test app features directly through the UI.
